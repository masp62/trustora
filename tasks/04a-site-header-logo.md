# #4a — Site Header & Logo

**Type:** AFK
**Blocked by:** #4

## What to build

Add a persistent site header to the root layout that appears on every page. The header contains the RealBnB logo/wordmark in the top-left corner, linking back to `/explore`. The header should be visually consistent with the existing amber/stone design system and remain fixed at the top of the viewport.

## Acceptance criteria

- [x] A `<header>` element is rendered on every page via the root layout
- [x] The RealBnB logo or wordmark is displayed in the top-left of the header
- [x] Clicking the logo navigates to `/explore`
- [x] The header uses the existing design tokens (amber/stone palette, heading font)
- [x] The header does not overlap page content (proper spacing/offset applied)
- [x] The header is responsive and works on mobile and desktop viewports

- #4 — HITL: Design Review — Auth & Onboarding Flow
